# VSIT Secured 🔒

VSIT Secured is a cybersecurity-focused website designed to help businesses protect their systems using zero trust principles and advanced IT management solutions.

## 🚀 Services

- Zero Trust Protection
- Application Control (ThreatLocker)
- Data Protection
- Patch Management (200+ applications)
- Remote IT Management

## 💡 About

This project is a lightweight, fast-loading website built using pure HTML, CSS, and JavaScript. It is designed for quick deployment and lead generation.

## 📊 Features

- High-converting cybersecurity landing page
- Security assessment quiz
- Service breakdown page
- Contact form

## 🛠 Tech Stack

- HTML5
- CSS3
- JavaScript

## 🔐 Security Focus

VSIT uses tools like ThreatLocker to enforce zero trust security, ensuring only approved applications can run.
